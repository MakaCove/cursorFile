# 覆盖维度参考

本页聚合了从需求文档、原型图中提取可测信息的方法，以及系统化的覆盖维度清单，用于发散思考时参考。

---

## 需求文档解析清单（PRD 维度）

拿到需求文档后，从以下角度提取可测信息：

### 功能模块拆分
- 一级模块与二级功能点
- 模块间的依赖关系与数据流
- 各模块的入口与出口

### 业务规则提取
- **约束类**：什么条件下允许/不允许某操作
- **计算类**：公式、汇率、税率、折扣规则等
- **权限类**：角色-操作映射表、数据权限范围
- **状态类**：状态枚举值、状态迁移条件、状态变更触发动作

### 校验规则提取
- **字段级**：必填、格式、长度、范围、唯一性
- **表单级**：字段间联动校验、条件必填
- **业务级**：跨表单/跨流程的业务校验

### 流程规则提取
- **正向流程**：主流程步骤与页面跳转
- **分支流程**：条件分支、可选步骤
- **异常流程**：超时、失败、回退、取消
- **审批流**：多级审批、会签/或签、驳回路径

### 数据规则提取
- **展示规则**：数据来源、排序、分组、格式化
- **导出规则**：导出范围、文件名、格式（Excel/PDF/CSV）
- **数据权限**：行级/列级权限、数据脱敏

---

## 原型解读清单（UI 维度）

- **导航**：面包屑、返回行为、路由参数、深链
- **搜索/筛选**：筛选条件（多选/单选/文本+模糊搜索）、**查询**与**重置**按钮、默认值、筛选标签、输入防抖
- **列表/表格**：分页、排序、列显隐/宽度、行选择、批量操作、**备注列及备注编辑入口**
- **表单**：必填标识、placeholder vs label、错误提示位置、帮助文案、自动保存
- **弹窗/抽屉**：关闭按钮、ESC、点击遮罩、未保存提示、焦点陷阱
- **Toast/提示**：时机、堆叠、重复提交去重
- **权限**：隐藏/禁用/点击后报错的策略一致性；**接口无权限时的返回**
- **空状态**：首次进入、筛选后无结果
- **加载**：骨架屏/转圈、重复刷新、取消请求
- **错误**：重试、离线、超时、服务端 4xx/5xx、部分失败

## 常见覆盖维度

以下为常见能力维度，**仅当原型或需求中确实存在时**才纳入用例设计，不必全部覆盖：

### 基础功能
- **新增**：正常新增、必填缺失、字段校验、唯一性校验、新增后列表刷新、新增后详情正确
- **编辑**：正常编辑、编辑校验同新增、不同状态下编辑、并发编辑冲突
- **删除**：单个删除、批量删除、删除确认、删除后列表刷新、关联数据影响
- **查看**：详情展示、列表展示、空数据展示、富文本/图片展示

### 列表能力
- **筛选/查询**：筛选条件（多选、单选、文本、模糊搜索等）→ 点击查询 → 列表展示符合条件数据
- **重置**：点击重置 → 筛选条件清空，列表刷新
- **分页**：首页、末页、翻页、改变每页条数、跳转到指定页
- **排序**：单列排序、多列排序、默认排序

### 流程能力
- **审批流**：提交审批、通过、驳回、撤回、多级审批流转
- **状态流转**：状态变更触发动作、状态变更校验、状态变更通知
- **导入导出**：导入文件格式校验、导入数据校验、导出范围与格式、导出文件名

### 权限能力
- **功能权限**：若需求提到"按菜单/按钮控制"，覆盖无权限时按钮可见性/可用性及提示
- **数据权限**：若需求提到数据权限，覆盖不同角色看到的数据范围
- **接口权限**：若需求提到接口鉴权，覆盖直接调接口时无权限的返回

### 数据能力
- **数据一致性**：多表关联更新、缓存与数据库一致性、事务回滚
- **操作日志/审计**：哪些操作写入日志、日志展示顺序与内容
- **消息通知**：触发条件、通知内容、通知方式（站内信/短信/邮件）、跳转链接

### 特殊场景
- **配置类页面**（多块 + 行动态增删）：每块必填项与默认值、行的添加与删除、**仅剩一条时删除置灰或不可删**（边界）、块内或跨行**唯一性校验**
- **报表/看板类页面**：数据来源与展示规则、树形/层级展示、多维度分行规则、字段计算公式、自动更新 vs 手动更新、导出规则
- **仪表盘/大屏**：实时刷新、数据下钻、图表交互、时间范围切换

## 风险节点识别清单

在分析需求与原型时，识别以下高风险节点，对所在用例自动提升优先级至 P0 并增加异常覆盖：

| 风险类型 | 识别特征 | 用例增强动作 |
|----------|----------|-------------|
| **外部系统依赖** | 支付网关、第三方API、短信/邮件服务、银行接口 | 增加超时、不可用、返回异常数据等异常用例 |
| **并发操作** | 多人同时编辑、库存抢购、秒杀、同时审批 | 增加并发冲突、数据一致性用例 |
| **数据一致性** | 多表关联写入、缓存更新、异步处理 | 增加事务回滚、缓存与DB一致性校验用例 |
| **超时/时效** | N分钟内支付、定时关单、Token过期 | 增加时间边界（刚好到期前/后）、时钟偏差用例 |
| **人工介入** | 人工审核、线下确认、手动导入 | 增加人工节点遗漏/延迟/错误的用例 |
| **金额/资金** | 支付、退款、结算、分账 | 增加金额精度、负数、零值、并发扣款用例 |
| **隐私/合规** | 个人敏感信息、数据脱敏、审计日志 | 增加权限校验、日志完整性、脱敏正确性用例 |
