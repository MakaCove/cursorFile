# 测试数据预设模板

生成用例时，可直接从以下模板中选取测试数据，避免每次重复构造。

---

## 边界值数据

| 数据类型 | min-1 | min | min+1 | 正常值 | max-1 | max | max+1 |
|----------|-------|-----|-------|--------|-------|-----|-------|
| 长度 2-20 字符 | "a" | "ab" | "abc" | "测试用户名" | 19 个字符 | 20 个字符 | 21 个字符 |
| 数值 1-100 | 0 | 1 | 2 | 50 | 99 | 100 | 101 |
| 金额 0.01-99999.99 | 0 | 0.01 | 0.02 | 1000.00 | 99999.98 | 99999.99 | 100000.00 |
| 分页条数 10-100 | 9 | 10 | 11 | 50 | 99 | 100 | 101 |

## 特殊字符

| 类别 | 测试数据 | 用途 |
|------|----------|------|
| HTML 标签 | `<script>alert(1)</script>` | XSS 测试 |
| HTML 实体 | `&lt;div&gt;` | 转义测试 |
| SQL 注入 | `' OR '1'='1` | SQL 注入测试 |
| SQL 注入（注释） | `admin'--` | SQL 注入变体 |
| 换行符 | `line1\nline2` | 多行输入 |
| Tab 字符 | `col1\tcol2` | 制表符 |
| 首尾空格 | `"  test  "` | 空格处理 |
| 全角字符 | `ＡＢＣ１２３` | 全角字符 |
| Emoji | `😀🎉❤️` | 特殊 Unicode |
| 零宽字符 | `​‌` | 零宽字符注入 |
| 右对齐字符 | `‮` | Unicode 方向控制 |
| 超长文本 | 10000 字符的 lorem ipsum | 字段溢出 |
| 路径遍历 | `../../../etc/passwd` | 路径注入 |
| NULL 字节 | `test\0value` | NULL 截断 |

## 常用格式

| 格式类型 | 有效示例 | 无效示例 |
|----------|----------|----------|
| 手机号（中国大陆） | `13800138000` | `12345`, `138001380000`, `abc` |
| 邮箱 | `test@example.com` | `test`, `test@`, `@example.com`, `test@.com` |
| 身份证（18位） | `11010119900101001X` | `12345`, `110101199001010011` |

| 日期（yyyy-MM-dd） | `2026-01-15` | `2026-13-01`, `01-15-2026`, `abc` |
| IPv4 | `192.168.1.1` | `256.0.0.1`, `192.168.1`, `abc` |
| 邮政编码 | `100000` | `1234`, `1234567`, `abcde` |

## 金额/数字

| 场景 | 测试数据 | 验证点 |
|------|----------|--------|
| 零值 | `0` | 零金额处理 |
| 负数 | `-100.00` | 负数校验 |
| 超大金额 | `999999999999.99` | 溢出/精度 |
| 多小数位 | `100.999` | 小数截断/四舍五入 |
| 科学计数法 | `1.5e3` | 格式校验 |
| 千分位 | `1,000.00` | 千分位解析/展示 |
| 前导零 | `000100.00` | 自动去除/保留 |

## 列表/分页数据

| 场景 | 数据条件 | 预期行为 |
|------|----------|----------|
| 空列表 | 0 条数据 | 展示空态 |
| 单条 | 1 条数据 | 正常展示，分页器不显示或置灰 |
| 刚好满页 | 20 条（默认每页 20） | 分页器不显示 |
| 满页+1 | 21 条 | 分页器出现，共 2 页 |
| 跨页操作 | 选中第 1 页某条，翻到第 2 页 | 第 1 页选中状态保留或清空（以需求为准） |
| 最后一页不足 | 总 25 条，每页 10 条 | 第 3 页展示 5 条 |
